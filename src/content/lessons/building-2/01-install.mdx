先说句大实话:**Claude Code 需要一个 Claude 付费账号**(Pro / Max 等),免费版用不了,而且国内访问不太方便。如果你嫌麻烦,直接用下面标注框里的 **Kimi Code** 更省事——用法几乎一样,这一关照着做就行。

装好后，在终端输入 <code style="background:#F4E2BE;padding:1px 7px;border-radius:6px;color:#9A6A2E;">claude --version</code> ，能看到一串版本号，就说明成功了。第一次运行 `claude` 它会让你在浏览器里登录一下，跟着提示点就行。

<Callout type="same">
**Windows** 把安装命令换成：<code>irm https://claude.ai/install.ps1 | iex</code>（在 PowerShell 里跑）。<br/>
想用 **Kimi Code**?安装命令是 <code>curl -LsSf https://code.kimi.com/install.sh | bash</code>，启动用 <code>kimi</code>，后面每一关都通用。
</Callout>
