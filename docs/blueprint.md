# AI 101 · 施工蓝图

> 给产品 / 运营 / 设计 / 测试等**非技术**的人,一条从 0 到 1、能真正上手用 AI 干活的路。
> 公益、免登录、图文并茂。

## 使命

不是"科普 AI 是怎么炼成的",而是**让一个怕 AI、没正经用过 AI 的人,真的能用 AI 把本职工作做完**,并理解"聊天 AI → Agent"这场转变。

## 11 条铁律(设计与写作都必须遵守)

1. **教学铁律**:每个概念必须绑一个 **before/after 演示**(截图/录屏/GIF),没有演示就不许出现。
2. **So-what 优先**:讲完"是什么",立刻讲"你能干嘛、下一步按哪个键"。拒绝只讲 What。
3. **结构**:线性阶梯为主干 + 术语卡为辅。双链只做"岸边注释",绝不做导航主干。
4. **单关模板**:一关一概念 · 四段式 · 2~3 分钟读完(约一屏多)。
   - ① 你现在的烦恼(共情钩子) ② 看个对比(before/after) ③ 它叫什么(此时才给术语 + 术语卡) ④ 你去试一下(现成动作/提示词)
5. **节奏**:碎关(拆零件)+ 合练关(组装真实任务);每 3~4 碎关 + 1 合练关;**首胜要早**(第1章第2关后就给迷你合练)。
6. **图**:v1 用截图/录屏/GIF + 极简示意图(宜家说明书,不是电路图,一图一意 ≤3 元素);后期升级原创插画/漫画。
7. **两栋楼**:楼① 聊天 AI(全民,完读即闭环) → 楼② Agent CLI 实操(进阶)。
8. **范围**:v1 只做这两栋楼;前沿 feed 是 v2,**物理隔离**(独立板块、可用双链网状结构)。
9. **底座**:Astro + 完全自定义设计系统(islands 撑任意交互组件)。
10. **调性**:温暖手账 + 轻闯关,一个憨憨吉祥物带路。
11. **工具一致性**:实操同屏只出一个工具(Claude Code);Kimi Code / 豆包 / DeepSeek / ChatGPT 只活在"📌 …也一样"标注框里,不进演示截图。

## 内容生产铁律

> **权威源(Claude/OpenAI 官方文档等)只提供"事实正确性",绝不提供"讲法和高度"。**
> 借它的料,绝不借它的复杂腔。抓到事实后,强制过四段式模板重写,再用"随便抓个产品/运营能否一句话复述"这个闸门自检,复述不出 = 重写。

## 章节骨架(v1)

### 🏠 楼① 聊天 AI(全民层,完读即闭环)

- **第0章 这站怎么用**(引导页):你能学到啥、怎么闯关、吉祥物登场
- **第1章 把话说清楚**(提示词)— 任务:写一条得体的通知
  - 越具体越准 → 给它角色 → 给它格式 →〔迷你合练:写条请假通知〕→ 给它一个例子
  - 开头标注框:📌 全程用 Kimi 示范,豆包/DeepSeek/ChatGPT 都一样
- **第2章 喂资料给它垫底**(上下文)— 任务:把会议记录变纪要
  - 粘资料 → 记不了太久(窗口)→ 新对话=清空 →〔合练:生成纪要〕
  - 内嵌便利贴:⚠️ 别喂这些(公司机密 / 客户隐私 / 未公开数据)
- **第3章 它会一本正经胡说**(幻觉/核对)— 任务:做一份"可信"的资料
  - 为啥编 → 让它老实 → 关键信息自己核 + 让它给出处 →〔合练:带核对的竞品资料〕
- **第4章 把 AI 当同事**(角色/拆步/迭代)— 任务:完成一件本职活
  - 扮专家 → 拆步骤 → 反馈迭代 → 让它先问你再动手 →〔合练:按岗位选一件真实产出〕
- **第5章 世界观升级:从"问答"到"它自己干活"**(Agent 是什么)— 看演示,不亲手
  - 聊天AI=问答,Agent=自己动手用工具 → 一个完整演示 → 为什么是趋势 →〔桥:通往楼②〕

### 🏗️ 楼② Agent CLI 实操(进阶层,Claude Code 为主)

- **第6章 装好你的第一个 Agent**(环境)— ⚠️ 全站最高危劝退关,手把手 + 报错急救
- **第7章 让它读懂你的活儿**(CLAUDE.md ≈ KIMI.md)— before/after 看它行为变化
- **第8章 给它装个专长:SKILL** — 此刻读者真能装、真能看到变化,SKILL 到这里才出现
- **第9章 合练:用 Agent 干完一件多步骤真活**(楼②总通关)
- **结尾页**:你已经会了啥 + 预告 v2 前沿情报站
